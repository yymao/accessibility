# What to Do If I Have a Long Slide Deck with Many Images?

Adding alt text to all the non-text content in a long slide deck can be time-consuming, but it is a required step to make the slides accessible.

## Use AI Tools

Save (or export) your existing slides as a PowerPoint file (`.pptx`).
Make sure there is no sensitive information in the file, and upload it to an AI chatbot.
Here is an example prompt you can use:

> Background: The attached file contains slides on <topic of the presentation> in an undergraduate-level <course subject> course.
>
> Task: Identify all non-text objects, including images, shapes, pictures, and charts, that are embedded in this file. For each object, inspect it and generate alternative text. Use the slide content as context when appropriate.
>
> Format: List the alt text for each object and the corresponding slide number in your output.
